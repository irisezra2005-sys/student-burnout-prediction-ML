# student-burnout-prediction-ML

## Project Overview
This project builds Machine Learning models to predict students Burnout Risk Level using AI student impact dataset. The project includes data preprocessing, model training, evaluation, hyperparameter tuning, and performance visualization.

## Dataset 
Dataset Name - AI Student Impact Dataset
Source - Dataset used in the previous EDA and preprocessing project. Also called cleaned dataset
Target Variable- Burnout_Risk_Level

## Machine Learning Models
- Logestic Regression
- Decision Tree Classifier

## Evaluation Metrices
- Accuracy
- precison
- Recall
- F1-Score

## Hyperparameter Tuning 
GridSearchCV was used to optimize the Decision Tree model and improve its perfomance.

## Results 
The models were compared using Accuracy, Precision, Recall, and F1-Score. Hyperparameter tuning was performed to improve the Decision Tree mode, and confusion matrices were generated for performance evaluation
